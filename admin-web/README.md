# Bordeaux Date Admin (Next.js)

Dashboard restaurant pour gérer les commandes et statuts.

## Démarrage
```bash
cd admin-web
npm run dev
```

## Variables d'environnement
Créer un fichier `.env.local` dans `admin-web/` :
```
ADMIN_PASSWORD=your_password
NEON_DATA_API_URL=https://ep-wispy-sun-akl0bvkr.apirest.c-3.us-west-2.aws.neon.tech/neondb/rest/v1
NEON_DATA_API_JWT=your_jwt
```

## Accès
Ouvrir `http://localhost:3000` → login → dashboard.
